Latest Project

Kittenbot OS Is A Open Source "Operating System" That Allows You To Store Multiple Games On Any Device Designed For Makecode Arcade Games! It Comes With A Calculator, Stopwatch, And Screensaver App, A Settings Menu To Change The Color Scheme And 3 Pre-Made Games (flappy bird, spaceships and ballistic balloons). There Is Also A Editable Version Without Games That You Can Add Your Own Games To!
